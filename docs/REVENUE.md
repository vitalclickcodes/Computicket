# Revenue Model

## 1. Commissions

Default platform commission per vertical (configurable per vendor):

| Vertical | Commission |
| --- | --- |
| Events & Concerts | 7–10% |
| Bus Travel | 4–6% |
| Flights | 1–3% + supplier markup |
| Hotels | 10–15% |
| Vouchers | 8–12% |

## 2. Convenience fees

Small per-ticket fee passed to the buyer (₦100–₦500 depending on order value). Transparent at checkout.

## 3. Vendor subscriptions

Optional tiered plans for organizers and operators:

- **Starter** — free, default commission
- **Growth** — monthly fee, lower commission, featured placement credits
- **Enterprise** — custom, dedicated account manager, API access, white-label scanner

## 4. Advertising

- Featured events on homepage and category pages
- Sponsored search results
- Email/push promo placements (with strict frequency caps)

## 5. Corporate solutions

- Enterprise booking dashboards (travel + events for staff)
- Central billing and invoicing
- Custom approval workflows

## 6. Float & wallet economics (Phase 2+)

- Interest on customer wallet balances (where regulation allows)
- FX margin on international flights and inbound tourism
