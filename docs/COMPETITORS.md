# Competitive Landscape

## Local (Nigeria)

| Competitor | Strength | Gap we can exploit |
| --- | --- | --- |
| Tix Africa | Strong with creator/event organizer self-serve | Single vertical — events only |
| Nairabox | Cinema, events, bills | UX and trust signals, no travel |
| Ariiya Tickets | Events focus, organizer tools | Limited verticals, less brand recognition |

## Adjacent Nigerian players

- **Bus**: GIGM, ABC, GUO each run their own bookings — fragmented, no unified search
- **Flights**: Wakanow, Travelstart — strong on flights/hotels, weak on events
- **Hotels**: Hotels.ng — hotels only

## Global inspiration

- **Ticketmaster** — scale, primary + resale, venue partnerships
- **Eventbrite** — organizer self-serve and discovery
- **Booking.com** — trust, reviews, conversion engineering
- **Expedia** — bundling across verticals

## Our wedge

The Nigerian market has strong single-vertical players but no credible unified booking ecosystem with first-class local payments, mobile-first UX, and anti-fraud ticketing. The bundle — events + bus + flights + hotels + vouchers + wallet — is the differentiator, and Phase 1 (events + bus) is the cheapest place to prove the loop.
